# AI Native Research Lab

AI Native 시대의 AX기획팀 전환을 연구하고, 작은 의사결정을 하나씩 쌓아가는 공유 연구실입니다.

## 목적

이 저장소는 완성된 정답을 보관하는 공간이 아닙니다.  
팀원들과 함께 현재 연구 상태를 공유하고, 의견을 모으고, 합의된 의사결정을 기록하기 위한 공간입니다.

## 현재 연구 주제

**AX기획팀의 기획 업무를 AI Native 방식으로 전환하는 방법**

## 운영 원칙

1. 한글을 기본 언어로 사용합니다.
2. 작은 의사결정을 하나씩 기록합니다.
3. 개인 의견과 팀 합의를 구분합니다.
4. 확정된 내용만 프레임워크로 정리합니다.
5. 특정 AI 도구나 벤더에 종속되지 않습니다.

## 역할 분리

| 공간 | 역할 |
|---|---|
| ChatGPT Project | 연구, 분석, 설계, 초안 작성 |
| Research Board | 팀 공유, 의견 수렴, 의사결정 기록 |
| Google Spreadsheet | 데이터의 단일 진실 원천(SSOT) |
| GitHub Pages | 팀원이 웹에서 확인하는 공유 연구실 |
